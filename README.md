# Qalam Helper

A simple Chrome extension that saves you hours of manual calculations on Qalam by showing you the remaining number of classes you can skip before getting an XF, and helping you track your performance across all courses by comparing your grades with the class average.

## Features

- **Attendance Tracking**: Shows how many more classes you can miss before hitting the XF thresholds
- **Class Comparison**: See how you're performing compared to the class average for every course 

## Installation

### Method 1: Chrome Web Store (Recommended)

Install directly from the Chrome Web Store: [Qalam Helper](https://chromewebstore.google.com/detail/qalam-helper/mhcphpancpjccfbpedemhjmilechbfhi)

1. Click the link above
2. Click "Add to Chrome"
3. Confirm by clicking "Add extension"
4. Visit [qalam.nust.edu.pk](https://qalam.nust.edu.pk) to see it in action!

### Method 2: Manual Installation (Unpacked)

1. **Download the Extension**
   - Click the green "Code" button above
   - Select "Download ZIP"
   - Extract the ZIP file to a folder on your computer

2. **Install in Chrome**
   - Open Chrome and go to `chrome://extensions/`
   - Enable "Developer mode" (toggle switch in top right corner)
   - Click "Load unpacked"
   - Select the extracted folder
   - Done! Visit [qalam.nust.edu.pk](https://qalam.nust.edu.pk) to see it in action

3. **Install in Edge**
   - Open Edge and go to `edge://extensions/`
   - Enable "Developer mode" (toggle on left sidebar)
   - Click "Load unpacked"
   - Select the extracted folder

4. **Install in Firefox**
   - Open Firefox and go to `about:debugging#/runtime/this-firefox`
   - Click "Load Temporary Add-on..."
   - Select the `manifest.json` file from the extracted folder

## Privacy
